# Memory Game

## Description

The Memory Game is a classic card-matching game implemented with JavaScript, HTML, and CSS. The goal is to match pairs of cards by flipping them over. This game includes features like dynamic difficulty, responsive design, and a restart button. Itâ€™s a great way to practice JavaScript and DOM manipulation skills.

## Features

- **Card Matching**: Flip cards to find matching pairs.
- **Dynamic Difficulty**: The game adjusts based on the number of cards.
- **Responsive Design**: Adapts to various screen sizes.
- **Restart Button**: Easily restart the game.
- **Game Status**: Displays a congratulatory message upon winning.

## Usage
1. Click on any card to flip it over.
2. Try to find matching pairs.
3. Once all pairs are matched, a congratulatory message will appear.
   
## Technologies Used
